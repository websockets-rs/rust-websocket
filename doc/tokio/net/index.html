TCP/UDP/Unix bindings for tokio.

This module contains the TCP/UDP/Unix networking types, similar to the standard library, which can be used to implement networking protocols.

Organization

  • TcpListener and TcpStream provide functionality for communication over TCP
  • UdpSocket and UdpFramed provide functionality for communication over UDP
  • UnixListener and UnixStream provide functionality for communication over a Unix Domain Socket (available on Unix only)
